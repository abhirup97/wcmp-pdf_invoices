WCMp PDF Invoices - A dynamic add-on for admin to automatically create and send PDF invoice to vendors.

== Description ==
<strong>WCMp PDF Invoices</strong> automatically generates PDF invoices of the orders placed for customers, vendors and admin as is the case. Additionally, vendors can also download Packing Slips for delivery convenience.
WCMp PDF Invoice  let’s the admin/ vendor decide their invoicing template from a range of 5 fully customizable invoice designs.☺
